| Symptom | Most likely fix |

| `pnpm build` fails | Usually a missing export → check `src/lib/core/index.ts` re-exports all new modules |
| `pnpm test --coverage` <70% | Add tests for uncovered branches → re-run |
| Spec is ambiguous or contradictory | Output `⛔ BLOCKED: <what's unclear>` and STOP — do not guess |
